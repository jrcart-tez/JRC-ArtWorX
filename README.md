# JRC ArtWorX

Fully deployable NFT minting app for Base + Farcaster.

## Features
- Upload art
- 1/1 or Editions
- Fixed price mint
- IPFS (NFT.Storage)
- Auto OpenSea collection: JRC ArtWorX
- Auto Farcaster Frame per drop
- Creator dashboard + gallery

## Deploy
1. npm install
2. Configure .env
3. Deploy contract
4. Deploy app
